# 프론트엔드 관리 문서

## 개요 및 목적

이 문서는 프로젝트의 프론트엔드 상태를 정의하고, 모든 팀원이 이를 명확하게 이해하는 것을 돕기 위해 작성되었습니다.

## 1. 전역과 지역 state

### 1-1 전역

    1. userState:
        1. 닉네임
        2. 이메일
        3. jwt 저장된 경로
        4. 장바구니
        - List<{item ID, item'name, item'price, item'option, item'quantity}>
        5. 주문할 매장 {매장ID, 매장명, 매장 위치}
        6. 진행중인 주문 List<{주문ID, 주문 상태, 주문한 아이템} >

### 1-2 지역

    1. 매장 정보
    2. item 정보
    3. 카드 잔액
    4. 과거 주문내역
